# Dotfiles and scripts I use on dev machines

> Based off https://github.com/haacked/dotfiles

## Setup

1. Clone repo ```git clone https://github.com/wicksipedia/dotfiles.git```
2. Run setup
   ```.\setup-choco.ps1```
3. Open PWSH (PowerShell 7)
   ```.\setup-shell.ps1```
4. Reload profile
   ```. $profile```

## What does this add?

Installs:

- Scoop
- Eartrumpet
- Git
- GitHub CLI
- Oh My Posh
- Linqpad
- Notepad++ (and redirects notepad to it)
- SysInternals
- Windows PowerToys
- Windows Terminal
- CaskaydiaCove NF
- Terminal-Icons

Customisations:

- Notepad - redirects to Notepad++
- Git
  - Aliases
- Powershell
  - Aliases
  - Hotkeys/Chords
